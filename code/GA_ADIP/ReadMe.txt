This folder contains all the relevant files for GA based tuning. 

The nomenclature follows a common theme: 
GA_LQi - The main program for LQi tuning
ga_params_LQi - The parameter file for LQi
fitnessfunc_LQi - The fitness function for LQi. 
GA_LQi_sim - Simulation model for LQi

Similarly for others.

centBest.mat - contains the 'Best' centres for RBFs considered.
